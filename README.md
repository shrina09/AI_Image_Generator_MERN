# AI_Image_Generator
## Description

This is an AI Image Generator website that generates an image based off of user prompt using OpenAI's API. Once the user Generates an Image it will be displayed in the Gallery, which shows all the images that are generated by which user and what prompt was used.

## Getting Started
### Dependencies

It requires Node.js and npm installed on the machine that will run this program. It also requires an API key from OpenAI API, MongoDB, and Cloudinary. In addition to the Cloudinary API key, it requires Cloudinary API secret and cloud name. There an example .env file provided called ".env_samples". Create a .env file and add the contents of the ".env_samples" file. Where it says "YOUR_API_KEY_HERE" type in the API key, where it says "YOUR_CLOUDINARY_CLOUD_NAME" with Cloudinary name and "YOUR_CLOUDINARY_API_SECRET" with the Cloudinary API secret. To obtain the API key, sign up to OpenAI, MongoDB, and Cloudinary.



### Executing program
* To set up
```
For frontend:
cd client
npm create vite@latest client --template vanilla
```

* How to build and run the program
```
To run the server:
cd server
npm start

To run the frontend:
cd client
npm run dev

On any browser of choice:
http://localhost:5173/
```
